# Summary material for CSDA Program Evaluations: Multispectral/Hyperspectral SME Groups

| Author | Affiliation |
|----------|----------|
| Paul Montesano, PhD    | NASA Goddard Space Flight Center     |

Commercial spaceborne image data are requested & purchased by CSDA, received, and organized. This organization is an iterative, complicated, yet necessary step to understand what images we receive for evaluation, and where. This summarization material is designed to provide SME teams with an organization set of information about the data we will work with, so that evaluations can be standardized and replicable. Here, we compile and update code to build:
1. prototype summaries of vendor image data that we receive to evaluate.
2. a database of image evaluation sites (AOIs) stored as a GeoJSON
