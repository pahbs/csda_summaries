# Summary material for CSDA Program Evaluations: Multispectral/Hyperspectral SME Groups

| Author | Affiliation |
|----------|----------|
| Paul Montesano, PhD    | NASA Goddard Space Flight Center     |

Commercial vendor spaceborne image data are requested & purchased by CSDA, received, and organized. This organization is an iterative, complicated, yet necessary step to understand what data we receive, and where. This summarization material is designed to provide SME teams with an organization set of information about the data we will work with, so that evaluations can be standardized and replicable. Here, we compile and update code to build prototype summaries of vendor data that we receive to evaluate.
